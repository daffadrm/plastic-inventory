# plastic-inventory-point-of-stock

## Getting started

**Plastic Inventory â€“ Point of Stock** dibangun menggunakan **Next.js**, **TypeScript**, dan **TailwindCSS**. Aplikasi ini digunakan untuk pencatatan stok, pengelolaan barang masuk/keluar, serta visualisasi data melalui dashboard interaktif.

## Persyaratan Sistem

Sebelum menjalankan aplikasi, pastikan Anda sudah menginstal:

- **Node.js 18+**
- **npm**
- Koneksi internet untuk instalasi dependency

## Instalasi & Menjalankan Aplikasi

Ikuti langkah berikut untuk menginstal dan menjalankan aplikasi.

```bash
npm install
```

setelah berhasil, jalankan dengan perintah berikut

```bash
npm run dev
```

lalu akses di browser

```bash
http://localhost:2207
```
